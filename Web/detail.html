我的兴趣：不只是“爱玩电脑”

这一页会更细致地介绍我在大数据、编程、游戏和音乐方面的兴趣，以及我做过的一些小尝试。

兴趣板块 · 详细版

仿照一些产品网站的“功能 Tab”，这里用选项卡的形式分块展示我的几个核心兴趣。 点击不同标签可以切换内容。

兴趣一：大数据运维 & 编程

我对大数据方向感兴趣，主要是喜欢折腾环境、解决报错的过程。 搭建 Hadoop、Hive、Spark、Flink、Kafka、ClickHouse 等组件， 再配合 Java / Python 写脚本，把数据串起来，是一件很有成就感的事。

  • 会在虚拟机中搭建多节点大数据集群。
  • 尝试用 Flink + Kafka + ClickHouse 做实时数据可视化。
  • 用 Java / Python 写爬虫和简单后端接口。

兴趣二：游戏 & CS:GO

游戏对我来说不仅是放松方式，也是观察数据的一个窗口。 例如在 CS:GO 中，不同地图、不同武器、不同皮肤价格都会形成数据， 这些都可以被采集、分析和可视化。

我做过一个简单的想法：抓取皮肤价格 → 写入 Kafka → 用 Flink 处理 → 存到 ClickHouse → 前端页面实时展示。

兴趣三：音乐 & 学习氛围

学习和写代码的时候，我很依赖背景音乐。 LoFi、纯音乐、电音、钢琴曲都在我的歌单里。 合适的音乐可以让时间过得更快，让复杂的配置过程没那么烦躁。

  • 准备了几个“学习专用”歌单，适合不同状态。
  • 喜欢在深夜安静的时候，一边听歌一边写代码。
  • 有时候会根据音乐节奏安排学习节奏。

兴趣四：大学生活 & 未来规划

在大学生活中，我希望自己不是“只会上课和刷手机”的那一种， 而是能够多做一点和未来职业相关的尝试，比如参加比赛、做小项目等。

我的规划是：先打牢基础，再让自己在一个方向上有比较完整的项目经历， 这样在毕业或者实习时就不会太慌。

兴趣成长之路

从最初的好奇到逐渐深入，这是我在兴趣爱好上的成长历程和未来规划。

阶段一：打好基础

熟悉 Java / Python 语法、Linux 基本命令和常用开发工具。

阶段二：大数据入门

了解并搭建 Hadoop、Hive、Spark、Flink、Kafka 等基础环境。

阶段三：小项目实战

做几个完整的小项目，例如日志分析平台、游戏价格可视化等。

阶段四：运维工程师方向

深入学习监控告警、性能优化、故障排查，向专业大数据运维工程师迈进。
